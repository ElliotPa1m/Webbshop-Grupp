# Webbshop-Grupp

### Verksamhetsbeskrivning
Spelbutik (typ som Epic Games)

Vi ska göra en webbshop där man kan köpa spel till datorn.

Pixelhuset.se

Målgrupp: Vi ska rikta oss mot spel intresserade i 15 till 30 års åldern.

Vi vill göra en webbshop som kan hjälpa folk hitta spel snabbt och enkelt. Ge tips till kunderna vad vi rekommenderar för spel och vad som är populärt! Vår kundkrets är intressarade av att hitta dom senaste spelen eller kunna söka upp ett specifikt spel.
